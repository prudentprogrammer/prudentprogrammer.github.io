CSTDM/MTC Transportation Project

The main goal of this project was to find the percentage of emissions/attributions and map them to specific regions in the Bay Area / Sacramento Area. For this project, I was given a file with 22 million trips and 8 million tours. I parsed the files by dumping the content onto a database and writing queries or python scripts to analyze the data. After the schemas were set up, I coded up the 5 various algorithms of finding percentages of attributions.

Languages/Frameworks used: Python, Psycopg2, PostgreSQL, PgAdmin, PostGIS, PgRouting, Google Charts
